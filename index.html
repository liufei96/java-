使用

1. 本地环境安装

  • 安装node.js
  • npm install gitbook-cli -g

2. 运行项目

  • git init 初始化一下项目
  • git install -y 安装下插件
  • git serve启动项目默认是4000端口
  • 启动之后访问localhost:4000
